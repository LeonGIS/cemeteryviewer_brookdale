Brookdale Cemetery

Online Help:

The Brookdale Cemetery Viewer provides the user burial information for the Brookdale Cemetery in Dedham, Massachusetts. It is a single application used on a desktop PC, mobile phone, and tablet device.

To locate a burial lot, simply click on the map or use the search box and select an item from the list.

Using the Application

This application is named the Brookdale Cemetery Viewer. It offers a map-based view of cemetery information that enables the user to locate lot and review burial information.

Finding an Area of Interest

There are three primary ways to identify an area of interest:

1.You can interact directly with the map and click on a burial lot when you've located it.
2. You can also use your current location within the cemetery.
3. You can search for a lot by an individual's name or by lot number .

Map Based Navigation

There are several ways to interact with the map:

1. Hold down the shift key and drag a box on the map to zoom in
2. Hold Down Ctrl+Shift to zoom out
3. Roll your mouse roller to zoom in/out
4. Drag the zoom slider to zoom in/out

Searching

The search box can be used to find an individual or lot number:

1. Enter a complete or part of a lot number or an individual's first or last name
2. Click on the item in the results list to locate burial lot on the map

Browser Compatibility

The Brookdale Cemetery Viewer application is supported on the following browsers: Microsoft IE 7/8/9, Mozilla Firefox, Google Chrome, Apple Safari, iOS Safari 4.0+, and Android Chrome 2.3+.
